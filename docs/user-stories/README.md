# User Stories - Royal Prestige

Este directorio contiene todas las **Historias de Usuario (HU)** documentadas para el proyecto Royal Prestige.

## Estructura

```
docs/user-stories/
├── README.md              # Este archivo
├── templates/
│   └── story-template.md  # Plantilla para nuevas HU
├── backlog/
│   ├── HU-001-sincronizar-carrusel.md
│   ├── HU-002-formulario-rifa.md
│   └── ...
└── index.md               # Índice de todas las HU
```

## Formato de nomenclatura

Cada historia de usuario se nombra siguiendo el patrón:

```
HU-XXX-descripcion-corta.md
```

Donde `XXX` es el número correlativo de la HU.

## Estados de una HU

- **Pendiente** - En el backlog, sin trabajar
- **En Progreso** - Actualmente en desarrollo
- **Completada** - Implementada y verificada
- **Cancelada** - Ya no es relevante
